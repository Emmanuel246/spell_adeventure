# Spell & Shine Adventure

Spell & Shine Adventure is an interactive, educational web game designed to help young learners practice spelling and language skills in a fun and engaging way. 

## Features

- **Student Portal**: An engaging entry point where children can enter their name and choose their very own hero avatar (like a dragon!) to begin their journey.
- **Interactive Map**: A visual adventure map that lets students navigate through different educational challenges.
- **Educational Gameplay**: Interactive levels designed to make learning pronunciation, spelling, and vocabulary an adventure rather than a chore.
- **Teacher Dashboard**: A dedicated portal for educators to track student progress, monitor performance, and manage the learning experience.

## Tech Stack

This project is built using modern web technologies:

- **React** (with React Router for navigation)
- **TypeScript** for type safety
- **Vite** for fast development and building
- **Tailwind CSS** & **shadcn-ui** for beautiful, responsive styling
- **React Query** for state and data fetching

## Getting Started

### Prerequisites
Make sure you have Node.js and npm installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone <YOUR_GIT_URL>
   ```

2. Navigate into the project directory:
   ```bash
   cd spell-shine-adventure-main
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

## Development and Deployment

- You can edit the frontend components seamlessly using your preferred IDE. 
- Deployment is straightforward using services like Vercel, Netlify, or Lovable.
